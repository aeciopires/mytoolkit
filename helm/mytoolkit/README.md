# mytoolkit

![Version: 1.0.0](https://img.shields.io/badge/Version-1.0.0-informational?style=flat-square) ![Type: application](https://img.shields.io/badge/Type-application-informational?style=flat-square) ![AppVersion: 1.0.0](https://img.shields.io/badge/AppVersion-1.0.0-informational?style=flat-square)

A suite of developer utilities exposed as a web app, REST API, and CLI.

**Homepage:** <https://github.com/aeciopires/mytoolkit>

## Installing

```bash
make helm-lint
make kind-load
make helm-install
```

Or directly with Helm:

```bash
helm upgrade --install mytoolkit helm/mytoolkit --namespace mytoolkit --create-namespace
```

## Uninstalling

```bash
make helm-uninstall
```
