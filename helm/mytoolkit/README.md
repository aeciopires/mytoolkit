## Installing

Access a Kubernetes cluster.

If you have [kind](https://kind.sigs.k8s.io/) installed, create a local cluster and install the helm chart using the following commands:

```bash
make helm-lint
make kind-load
make helm-install
```

Or directly with Helm:

```bash
helm upgrade --install mytoolkit -f helm/mytoolkit/values.yaml helm/mytoolkit --namespace mytoolkit --create-namespace
```

Create a port-forward with the follow command:

```bash
kubectl port-forward svc/mytoolkit 8080:8080 -n mytoolkit
```

Open the browser and access the URL: http://localhost:8080

## Uninstalling

```bash
helm uninstall mytoolkit --namespace mytoolkit
```

Or with the Makefile:

```bash
make helm-uninstall
```
